BadmintonPro Setup

กรุณาตั้งค่า Firebase Config ก่อนใช้งาน

📋 วิธีตั้งค่า (~5 นาที)

1. ไปที่ console.firebase.google.com → สร้าง Project

2. Authentication → Sign-in method → เปิด Email/Password

3. Realtime Database → Create → Asia (singapore) → Test mode

4. Rules → แทนที่ด้วย: {"rules":{".read":"auth!=null",".write":"auth!=null"}}

5. Project Settings ⚙️ → เพิ่ม Web App → คัดลอก Config

6. แก้ไข FIREBASE_CONFIG ในไฟล์ HTML

7. Authentication → Users → เพิ่ม 3 accounts:

EmailPasswordRole
admin@badminton.proAdmin@1234admin 🔑
manager@badminton.proManager@1234manager 📊
officer@badminton.proOfficer@1234officer 🎫

* เปลี่ยน email/password ตามต้องการ แล้วแก้ ROLE_MAP ในไฟล์ด้วย
